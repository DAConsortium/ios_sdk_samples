## DAC NativeAd SDK

Version 0.1

### Overview

DAC NativeAd SDKはネイティブ広告をアプリ内に表示するSDKです。

本SDKはYieldOneのネイティブ広告のプレースメントIDを設定することにより、ネイティブ広告のレスポンスを得ることができます。

### Requirement

- Xcode 7.3+
- iOS 8.0+

### Install

Sample内、manual.mdをご参照ください。
