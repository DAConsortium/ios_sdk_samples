  • send(_:)

    アプリケーション情報をサーバーにリクエストします。

    • setup()時にautoSendApplicationDataを有効にしている場合、アプリがバックラグラウンドに遷移した際に自動的に実行されます。

    • delegateが有効な場合、先にdelegateに通知したのちにcompletionHandlerが実行されます。

    Declaration

    Swift

    public func send(_ completionHandler: ((_ error: Error?) -> Void)? = nil)

    Parameters

    completionHandler

    リクエスト完了後に起動するクロージャーです。リクエスト時にエラーが発生した場合、それがErrorの引数として渡されます。エラーコードについては、DACSDKPingVErrorCodeを参照ください。
