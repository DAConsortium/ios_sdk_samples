## DAC Ads SDK

Version 1.2.4


### Overview

DAC Ads SDKはバナー広告をアプリ内に表示するSDKです。

本SDKはYieldOneのバナー広告のプレースメントIDを設定することにより、バナー広告を表示することができます。


### Requirement

- Xcode 8.2+
- iOS 8.0+


### Install
Sampleのmanual.mdをご参照ください。
